# NeuralBooster Fix
BepInEx plugin for Casualties: Unknown that prevents the neural booster item from annihilating your eyeballs on subsequent uses, while keeping all the other downsides intact. Includes diminished scaling to balance it.

# Installation
## Installing BepInEx

1. Install BepInEx from https://github.com/BepInEx/BepInEx/releases (latest v5 release should be fine)
2. Extract all files into the folder containing CasualtiesUnknown.exe. Replace all files when prompted
   * Usually at `C:\Program Files (x86)\Steam\steamapps\common\Casualties Unknown Demo\CasualtiesUnknownDemo` on Steam
3. Run the game once to generate configs, then close it

## Installing the mod

1. Find your Plugins folder (`BepInEx/Plugins`)
2. Install the latest `.dll` from https://github.com/cetteon/NeuralBoosterFix/releases
3. Move the file you just downloaded to the Plugins folder
